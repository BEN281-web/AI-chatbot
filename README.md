# AI Chatbot

## What I Built
A chatbot using Node.js backend and vanilla JS frontend.

## API and Model
API: OpenRouter  
Model: openai/gpt-4o-mini  

Why backend:
API key in frontend can be stolen via browser DevTools. Backend keeps it secure.

Fallback:
Gemini API  
Change:
- Base URL
- Model name
